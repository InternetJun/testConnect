2, dist[n]表示了n个点，有一个点是固定的！既有dist[k-1] = 0;需要的是boolean used来表示出
~~~ java
	int[] dist = new int[n];
	Arrays.fill(dist,INF);
	dist[k-1] = 0;
	boolean[] used = new boolean[n];
	for (int i = 0; i < n; i++) {
		int x = -1;
		for (int y = 0; y < n; y++) {
			if (!used[y] && (x == -1 || dist[y] < dist[x])) {
				x = y;
			}
		}
		used[x] = true;
		for (int j = 0; j < n; j++) {
			dist[j] = Math.min(dist[j], dist[x] + g[x][j]);
		}
	}
~~~
