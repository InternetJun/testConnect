## 对单词搜索思考
1.自己去做的是对数据的依次比较，但是有这样的做法是最高的复杂度。
2.可以知道的是对于小写字母的26个进行比较的话，复杂度可以忽略不计的。
3.对于List内的元素判断的话需要转换Set，因为set内有contains的method，时间复杂度是O（1）。
